1. ADMIN NOTIFICATIONS

2. USER NOTIFICATIONS
   ✅ User nhận thông báo khi bị cảnh báo
   ✅ User nhận thông báo khi bị tạm khóa (mute)
   ✅ User nhận thông báo khi bị cấm vĩnh viễn (ban)
   ✅ Notification hiển thị trên chuông thông báo (header)
   ✅ Click notification → chuyển đến trang violations

3. NOTIFICATION TYPES
   ✅ REPORT_SUBMITTED (📋) - Admin nhận khi có báo cáo
   ✅ USER_WARNED (⚠️) - User nhận khi bị cảnh báo
   ✅ USER_MUTED (🔇) - User nhận khi bị tạm khóa
   ✅ USER_BANNED (🚫) - User nhận khi bị cấm
   ✅ EXAM_PUBLISHED (📝) - Student nhận khi có đề thi mới
   ✅ ATTEMPT_GRADED (✅) - Student nhận khi bài thi được chấm
   ✅ ESSAY_GRADED (✅) - Student nhận khi câu tự luận được chấm
   ✅ ESSAY_PENDING (📋) - Teacher nhận khi có bài tự luận chờ chấm

   ✅ Notification Bell Component
   ✅ Dropdown hiển thị danh sách notification
   ✅ Badge hiển thị số notification chưa đọc
   ✅ Click notification → navigate to link
   ✅ Unread indicator (blue dot)
   ✅ Polling mỗi 30 giây

5. BACKEND FEATURES
   ✅ NotificationService - Gửi notification
   ✅ NotificationController - API endpoints
   ✅ DiscussionReportService - Notification khi báo cáo
   ✅ AdminModerationService - Notification khi cảnh báo

6. API ENDPOINTS

================================================================================
                              TROUBLESHOOTING
================================================================================

ISSUE: Admin không nhận notification
  CHECK 1: Admin user có role ADMIN?
    SELECT * FROM users u 
    JOIN user_roles ur ON u.id = ur.user_id 
    JOIN roles r ON ur.role_id = r.id 
    WHERE r.name = 'ADMIN';
  
  CHECK 2: Backend logs có debug message?
    [DEBUG] Found X admins
    [DEBUG] Sending notification to admin: ...
  
  CHECK 3: Database có notification?
    SELECT * FROM notifications ORDER BY created_at DESC LIMIT 10;

ISSUE: Frontend notification bell không hiển thị
  CHECK 1: Browser console có error?
    F12 → Console tab
  
  CHECK 2: API trả về data?
    GET http://localhost:8081/api/notifications/unread-count
  
  CHECK 3: NotificationBell được import?
    import NotificationBell from './NotificationBell'
